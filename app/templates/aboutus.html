About Us

CryptoSeal is a project developed for CSAC 329 - Applied Cryptography (2024-25-2) at the Camarines Sur Polytechnic Colleges.
Our mission is to make cryptography accessible and practical for everyone, providing a secure, user-friendly platform for encryption, decryption, and hashing of both text and files.

We believe in the importance of data privacy and security in the digital age. This project demonstrates modern cryptographic techniques and best practices, integrating real-world features like user roles, address management, and rate limiting.

Encryption

Encrypt and decrypt text and files using modern symmetric and asymmetric algorithms.

Hashing

Hash text and files for integrity and verification using secure hash functions.

User Roles

Role-based access, address management, and rate limiting for security and usability.
